# Laravel CRUD 基礎實作 (windows)

### 安裝環境設定跟laravel專案

laravel安裝環境如下:

PHP >= 7.0.0<br>
OpenSSL PHP Extension<br>
PDO PHP Extension<br>
Mbstring PHP Extension<br>
Tokenizer PHP Extension<br>
XML PHP Extension<br>
Composer (PHP Dependency Manager)<br>
